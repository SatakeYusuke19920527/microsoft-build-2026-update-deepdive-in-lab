# 07. パートナーエージェントアプリ

**Partner-built agent apps** は GitHub Copilot と統合し、タスクの自動化・コード生成・コンテキスト分析・アクション実行を支援します。**GitHub を離れずに、お気に入りのツールを利用** できます。

## できること

- Issue を、自分のワークフローに合った **新しいエージェントに割り当て**。
- パートナー製エージェントを **今すぐ利用開始** できる。
- 自社のエージェントアプリを GitHub に持ち込むための **ウェイトリスト** に参加可能。
